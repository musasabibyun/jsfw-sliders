# Features

以下の機能をもったスライダーを， jQuery と Vue.js を用いて 2 つ作成しました．

**スライダーの機能一覧**
- ナビゲーションボタン (前後のスライドに移動させる)
- インジケーターボタン (任意のスライドに移動させる)
- 自動再生
- 一時停止

# Requirement
実装時 (2022/06 時点) のバージョン

jQuery: 3.6.0
Vue.js: 3.2.36
